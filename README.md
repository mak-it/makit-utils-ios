# MakitUtils

## Installation

MakitUtils is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MakitUtils', git: 'https://github.com/mak-it/makit-utils-ios.git', branch: 'master'
```
